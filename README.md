# System Design Playbook

Documentacao aberta sobre raciocinio, comunicacao visual e boas praticas de system design para entrevistas tecnicas.

## Colaboracao

Contribuicoes da comunidade sao bem-vindas, especialmente em:

- clareza editorial
- correcoes tecnicas
- exemplos e diagramas
- estrutura de navegacao e aprendizado

Antes de contribuir, consulte:

- [CONTRIBUTING.md](CONTRIBUTING.md)
- [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
- [SECURITY.md](SECURITY.md)
- [SUPPORT.md](SUPPORT.md)

## Seguranca

Se voce identificar uma vulnerabilidade ou exposicao acidental de segredo, nao abra issue publica. Siga as orientacoes em [SECURITY.md](SECURITY.md).
